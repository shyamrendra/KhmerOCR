## KhmerOCR

A Fast Khmer Optical Character Recognition. The model was trained on 3M text lines with Khmer 800+ fonts.

### Install

```shell
pip install -e .
```

### Usage

```shell
python recognize.py image.jpg
```

**Input Image**

<img src=image.jpg >
